#  Corpo Event

**Corpo Event** est une plateforme interne d'entreprise permettant de **partager des événements**, **proposer de nouveaux événements** et **interagir via des commentaires**. Elle facilite la communication entre les collaborateurs répartis sur différents sites.

---

##  Objectifs du projet

- Développer un système simple pour consulter, proposer et gérer des événements d’entreprise.
- Améliorer la diffusion de l’information en interne.
- Appliquer les principes de la POO (programmation orientée objet) et les bonnes pratiques SOLID.

---

##  Technologies et concepts utilisés

- **Java** – langage principal
- **POO (Programmation Orientée Objet)**
- **UML** – Use Case, Diagramme de Classes, Diagramme de Séquence
- **Git** – gestion de version

---

## Fonctionnalités principales

- 👤 Connexion des utilisateurs
- 📅 Consultation d'événements
- 💬 Ajout de commentaires sur les événements
- 🛡️ Interface d’administration pour la gestion des utilisateurs et événements
