## Règles de gestion

### Collaborateurs

- R1 : Un collaborateur doit avoir une adresse mail
- R2 : Un collaborateur doit avoir un mot de passe
- R3 : Un collaborateur doit avoir un nom
- R4 : Un collaborateur doit avoir un prénom
- R5 : Un collaborateur peut lire un événement
- R6 : Un collaborateur peut créer un commentaire
- R7 : Un collaborateur peut lire un commentaire
- R8 : Un collaborateur peut modifier uniquement un commentaire qui lui appartient
- R9 : Un collaborateur peut supprimer uniquement un commentaire qui lui appartient
- R10 : Un collaborateur peut participer à un évenement

### Administrateurs

- R11 : Un administrateur doit avoir une adresse mail
- R12 : Un administrateur doit avoir un mot de passe
- R13 : Un administrateur doit avoir un nom
- R14 : Un administrateur doit avoir un prénom
- R15 : Un administrateur peut lire un évenement
- R16 : Un administrateur peut créer un évenement
- R17: Un administrateur peut modifier un événement
- R18: Un administrateur peut supprimer un événement
- R19: Un administrateur peut créer un commentaire
- R20: Un administrateur peut lire un commentaire
- R21: Un administrateur peut modifier un commentaire
- R22: Un administrateur peut supprimer un commentaire
- R23: Un administrateur peut ajouter un collaborateur à un événement

### Evenements

- R24: Un événement doit avoir un intitulé
- R25: Un événement doit avoir une date de création
- R26: Un événement doit avoir une date de modification
- R27: Un événement doit avoir une description
- R28: Un événement peut avoir un ou des commentaires
- R29: Un événement peut avoir un ou des participants

### Commentaires

- R30: Un commentaire doit avoir un auteur
- R31: Un commentaire doit avoir un texte
- R32: Un commentaire doit avoir une date de création
- R33: Un commentaire doit avoir une date de modification
